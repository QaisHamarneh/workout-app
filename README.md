# Workout Timer

## Installation

Either download the .zip file or run `git clone https://github.com/QaisHamarneh/workout-app.git`.

## Setup

We recommend starting a new virtual environment: 

```
python3 -m venv .env
source .env/bin/activate
pip3 install -r requirements.txt
```

## Run

To run the code, simply run `python3 main.py`.
